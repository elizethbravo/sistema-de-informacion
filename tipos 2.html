TIPOS DE SISTEMA DE INFORMACION,SEGUN LOS NIVELES DE DECISION O ADMINISTRATIVO

se sintetizan algunas de las carcteristicas principales de los tipos de sistema de informacion segun los niveles de decision o administrativo.


existen 7 tipos de sistema los cuales son:
ESS DSS MIS IAS KWS OAS-office automation systems TPS
Sistemas de Soporte Ejecutivo,estos sistemas proporcionan a los directivos y ejecutivos información clave y herramientas para la toma de decisiones estratégicas dentro de una organización. Estos sistemas están diseñados para ayudar a los usuarios a tomar decisiones basadas en datos y modelos analíticos. Proporcionan herramientas para la recolección, organización y análisis de datos,facilitando así la toma de decisiones efectivas. Estos sistemas están diseñados para ayudar a las organizaciones a recopilar, almacenar, procesar y presentar información para apoyar la toma de decisiones,el control y la coordinación dentro de una empresa. Los MIS suelen generar informes y resúmenes que son útiles para la gestión empresarial Es un sistema diseñado para ayudar en la toma de decisiones y el control de procesos administrativos en una organización.también puede referirse a Sistema de Información Automatizados, que se enfoca en la automatización de procesos y la gestión de datos Estos sistemas están diseñados para ayudar a los trabajadores del conocimiento, como profesionales, investigadores y diseñadores, a crear y compartir nuevo conocimiento dentro de una organización Estos sistemas están diseñados para mejorar la eficiencia y productividad en las tareas rutinarias de oficina, como procesamiento de texto, gestión de documentos, comunicaciones internas, agenda electrónica, entre otros. Estos sistemas se utilizan para procesar y gestionar transacciones comerciales, como ventas, compras, pagos y otras operaciones diarias dentro de una organización.Los TPS son fundamentales para el funcionamiento eficiente de muchas empresas
